# ORAS (oras)
ORAS (OCI Registry As Storage) is a CNCF project that provides a CLI and a set of client libraries for pushing and pulling arbitrary OCI artifacts to and from OCI-compliant registries, allowing container registries to be used as a generic artifact distribution mechanism.

## APIs

### ORAS CLI
Generic command-line registry client used to push and pull OCI artifacts to and from any OCI-compliant container registry.

- [Documentation](https://oras.land/docs/)
- [Installation](https://oras.land/docs/installation)
- [Commands](https://oras.land/docs/commands/oras)
- [Source Code](https://github.com/oras-project/oras)

### ORAS Client Libraries
Client libraries for building custom OCI artifact tools and integrations on top of ORAS, available across multiple language ecosystems including Go, Rust, Python, JavaScript, .NET, and Java.

- [Documentation](https://oras.land/docs/client_libraries/overview)
- [Go](https://github.com/oras-project/oras-go)
- [Rust](https://github.com/oras-project/rust-oci-client)
- [Python](https://github.com/oras-project/oras-py)
- [JavaScript](https://github.com/oras-project/oras-js)
